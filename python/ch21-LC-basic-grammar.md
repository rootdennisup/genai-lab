## 1 变量与类型特性
- **动态类型**：Python 变量不需要预先声明，每个变量在使用前都必须赋值，变量赋值后即被创建（**动态类型**）。
- **内存本质**：变量就是变量，它没有类型，我们所说的类型是变量所指的内存中对象的类型。
- **类型查看**：使用 ``type()`` 函数查看变量当前的类型。

 **type vs. isinstance**：
- ``type()`` ：严格的身份鉴定，返回对象的精确类
- ``isinstance()`` ：家族关系的确认，检查对象是否属于某个类，或是否属于该类的派生类（子类）。符合面向对象设计中的 “是一个 (is-a)” 原则。


## 2 数据类型
- 数据类型是构建程序的地基。Python 属于强类型、动态类型语言，这意味着变量本身没有类型，类型属于内存中的对象。
- 我们将数据类型分为标准内置类型、类型转换规则以及现代工程中的高级扩展进行说明。

## 2 标准内置类型

### 2.1 数字 & 布尔

#### Number(数字)
- **构成**：Python3 支持 int（长整型）、float（浮点数）、bool（布尔）和 complex（复数）。

- **主要特点**：
  - Python 可以同时为多个变量赋值，如 a, b = 1, 2。
  - 一个变量可以通过赋值指向不同类型的对象。
  - 数值的除法包含两个运算符：/ 返回一个浮点数，// 返回一个整数（向下取整）。
  - 在混合计算时，Python 会把整型自动转换为浮点数。

#### bool(布尔)
- bool 是 **int 的子类**，因此布尔值可以被看作整数来使用，True 等价于 1，False 等价于 0。
  
- **主要特点**：
  - 布尔类型只有两个值：True 和 False。
  - 布尔类型可以和**其他数据类型**进行比较，如数字、字符串等。比较时，Python 会将 True 视为 1，False 视为 0。
  - 布尔类型可以和**逻辑运算符**一起使用，包括 and、or 和 not，用来组合多个布尔表达式。
  - 布尔类型可以被**转换成其他数据类型**，如整数、浮点数和字符串。转换时，True 会被转换成 1，False 会被转换成 0。
  - 可以使用 ``bool()`` 函数将其他类型的值转换为布尔值。
      - 以下值转换为布尔值时为 ``False``：None、False、零（0、0.0、0j）、空序列（如 ''、()、[]）和空映射（如 {}）。
      - 其他所有值转换为布尔值时均为 ``True``。

### 2.2 序列类型 (Sequences)

#### String（字符串）
- Python 中的字符串用单引号 ' 或双引号 " 括起来。字符串是**不可变类型**，支持双向索引（左起 0，右起 -1）和切片（[头:尾]）。

- **字符串切片**语法：``变量[头下标:尾下标]``

- **其他特点**：
  - 双向索引：索引值以 0 为开始值，-1 为从末尾开始的位置。
  - 反斜杠可以用来转义，使用 r 前缀可以让反斜杠不发生转义（原始字符串）。
  - 字符串可以用 + 运算符连接，用 * 运算符重复。
  - Python 中的字符串有**两种索引方式**：
    - 从左往右以 0 开始。
    - 从右往左以 -1 开始。

#### List（列表）
- 列表中元素的类型可以不相同，它支持数字、字符串、嵌套列表，是**可变类型**。
- 列表可以完成大多数集合类的数据结构实现，常用于实现堆栈（LIFO）或队列（FIFO）数据结构。

- **列表截取**的语法格式：``变量[头下标:尾下标]``

- **其他特点**：
  - 列表写在方括号 [] 之间，用逗号分隔开的元素列表。
  - 和字符串一样，列表也可以被索引和截取，列表被截取后返回一个包含所需元素的新列表。
  - 列表可以使用 + 操作符进行拼接。

#### Tuple（元组）
- 元组（tuple）与列表类似，不同之处在于元组的元素不能修改（不可变类型）。
- 元组写在小括号 () 里，元素之间用逗号隔开。
- 常用于存储异质数据集，或作为字典的键（Key）

- **其他特点**：
  - 元组也可以被索引和切片，方法与列表相同。
  - 注意构造包含 0 或 1 个元素的元组的特殊语法规则。
  - 元组也可以使用 + 操作符进行拼接。

#### bytes (字节)
- bytes 类型是不可变的二进制序列（byte sequence）。
- 与字符串类型不同的是，bytes 类型中的元素是整数值（0 到 255 之间的整数），而不是 Unicode 字符。
- bytes 常用于处理二进制数据，如网络传输，图像处理、音视频文件处理等。
- 创建 bytes 对象最常见的方式是使用 b 前缀：``x = b"hello" ``。

### 2.3 集合 & 字典

### Set（集合）
- Python 中的集合（Set）是一种无序、存储唯一元素的、**可变数据类型**。
- 集合中的元素不会重复，常用于去重或进行交、并、差集等数学运算。
- 集合使用大括号 {} 表示，元素之间用逗号 ``,`` 分隔。也可以使用 set() 函数创建集合。
- 创建一个空集合必须用 set() 而不是 {}，因为 {} 创建的是一个空字典。

### Dictionary(字典)
- 字典（dictionary）是一种映射类型，用 {} 标识，它是一个 **键(key) : 值(value) 的集合**。
- 键(key) 必须使用不可变类型，且在同一个字典中键必须是唯一的。

- **其他特点**：
  - 创建空字典使用 {}。
  - Python 3.7 起字典保持插入顺序。
  - 构造函数 dict() 可以直接从键值对序列中构建字典。


## 3 类型转换
- **隐式转换**：自动将一种数据类型转换为另一种数据类型。
  - 在混合运算中（两种不同类型的数据），Python 会自动将低精度（如 int）转换为高精度（如 float）以防数据丢失。

- **显式转换**：将对象的数据类型转换为所需的数据类型。
  - 使用 int()、float()、str() 等**预定义函数**来强制转换。


## 4 Python3 运算符

### 4.1 算术运算符
假设变量 a=10，变量 b=21

| 运算符 | 描述                                           | 例子               |
| ------ | ---------------------------------------------- | ------------------ |
| +      | 加，两个对象相加                               |                    |
| -      | 减，得到负数或是一个数减去另一个数             |                    |
| *      | 乘，两个数相乘或是返回一个被重复若干次的字符串 | a * b 输出结果 210 |

### 4.2 比较（关系）运算符
Python 比较运算符有：``==、!=、>、<、>=、<=``

### 4.3 赋值运算符
假设变量 a=10，变量 b=21

| 运算符 | 描述             | 例子                      |
| ------ | ---------------- | ------------------------- |
| =      | 简单的赋值运算符 |                           |
| +=     | 加法赋值运算符   | c += a 等效于 c = c + a   |
| -=     | 减法...          |                           |
| *=     | 乘法...          |                           |
| /=     | 除法...          |                           |
| %=     | 取模...          |                           |
| **=    | 幂  ...          | c **= a 等效于 c = c ** a |
| //=    | 取整除...        |                           |
| :=     | 海象运算符       |                           |

Python 3.8 版本引入海象运算符，用于**表达式中同时进行赋值和返回赋值的值**。在需要在表达式中使用赋值结果等情况下，可以起到简化代码的作用，示例：
```python
# 传统写法
n = 10
if n > 5:
    print(n)

# 使用海象运算符
if (n := 10) > 5:
    print(n)
```

### 4.4 成员运算符
- ``in`` ：如果在指定的序列中找到值返回 True，否则 False。
- ``not in``：如果在指定的序列中没有找到值返回 True，否则 False。

### 4.5 身份运算符
用于比较两个对象的存储单元。
- ``is``：判断两个标识符是不是引用自一个对象。
    - 示例： x is y, 类似 id(x) == id(y) , 如果引用的是同一个对象则返回 True，否则 False。
- ``is not``：判断两个标识符是不是引用自不同对象。
    - 示例：x is not y ， 类似 id(x) != id(y)。如果引用的不是同一个对象则返回结果 True，否则 False。

### 4.6 运算符优先级
| 运算符                                                               | 描述                               |
| -------------------------------------------------------------------- | ---------------------------------- |
| (expressions...),[expressions...], {key: value...}, {expressions...} | 括号的表达式                       |
| x[index], x[index:index], x(arguments...), x.attribute               | 读取，切片，调用，属性引用         |
| await x                                                              | await 表达式                       |
| **                                                                   | 乘方(指数)                         |
| +x, -x, ~x                                                           | 正，负，按位非 NOT                 |
| *, @, /, //, %                                                       | 乘，矩阵乘，除，整除，取余          |
| +, -                                                                 | 加和减                             |
| <<, >>                                                               | 移位                               |
| &                                                                    | 按位与 AND                         |
| ^                                                                    | 按位异或 XOR                       |
| \\|                                                                  | 按位或 OR                          |
| in,not in, is,is not, <, <=, >, >=, !=, ==                           | 比较运算，包括成员检测和标识号检测    |

- 如上表格列出了**从最高到最低**优先级的所有运算符，相同单元格内的运算符具有相同优先级。
- 运算符均指二元运算，除非特别指出。 相同单元格内的运算符**从左至右分组**（除了幂运算是从右至左分组）。

## 5 条件判断
通过一条或多条语句的执行结果（True 或 False）来决定执行的代码块。

条件判断关键字
| 关键字 / 函数 | 说明                                        | 示例                 |
| ------------- | ------------------------------------------- | -------------------- |
| if            | 条件判断语句，当条件为 True 时执行代码块    | if x > 0:            |
| elif          | 多条件判断分支（else if）                   | elif x == 0:         |
| else          | 所有条件不满足时执行                        | else:                |
| pass          | 空语句，占位用，保证语法完整                | if x > 0: pass       |
| match         | 结构化模式匹配（Python 3.10+，类似 switch） | match x: case 1: ... |

说明：
- 每个条件后面要使用冒号 :，表示接下来是满足条件后要执行的语句块。
- 使用缩进来划分语句块，相同缩进数的语句在一起组成一个语句块。
- Python 中没有 switch...case 语句，但 Python3.10 版本添加了 match...case，功能也类似。

### if 语句
```python
if condition_1:
    statement_block_1
elif condition_2:
    statement_block_2
else:
    statement_block_3
```

### match...case
```text
match subject:
    case <pattern_1>:
        <action_1>
    case <pattern_2>:
        <action_2>
    case <pattern_3>:
        <action_3>
    case _:
        <action_wildcard>
```
- case _: 类似于 C 和 Java 中的 default:，当其他 case 都无法匹配时，匹配这条。


## 6 循环语句
Python 中的循环语句有 ``for`` 和 ``while``。

循环控制关键字与方法:
| 关键字 / 函数 | 说明                                | 示例                                 |
| ------------- | ----------------------------------- | ------------------------------------ |
| for           | 迭代循环，用于遍历序列或可迭代对象  | for i in list:                       |
| while         | 条件循环，条件为 True 时持续执行    | while x > 0:                         |
| break         | 立即终止当前循环                    | else:                                |
| continue      | 条件循环，条件为 True 时持续执行    | 跳过本次循环剩余代码，进入下一次迭代 |
| else（循环）  | 循环正常结束（未被 break）时执行    | for i in range(3): ... else: ...     |
| pass          | 循环中的占位语句（空操作）          | for i in range(5): pass              |
| range()       | 生成整数序列，常与 for 循环配合使用 | range(0, 5)                          |
| enumerate()   | 遍历时同时获取索引和值              | for i, v in enumerate(list):         |

### while 循环
```python
while 判断条件(condition)：
    执行语句(statements)……
```

如果 while 后面的条件语句为 false 时，则执行 else 的语句块：
```python
while <expr>:
    <statement(s)>
else:
    <additional_statement(s)>
```

### for 语句
```python
for <variable> in <sequence>:
    <statements>
else:
    <statements>
```

## 7 正则表达式 vs. 数据清洗
如果说 [String（字符串）](#2-标准内置类型)是处理文本信息的地基，那么正则表达式 (Regex) 则是进行高效数据清洗与校验的“手术刀”。

### 7.1 re 模块
Python 通过标准库中的 re 模块 提供正则表达式（Regex）支持，主要用于文本的搜索、替换和分割。

**常用功能：**
- **文本搜索**：匹配特定的字符模式。
- **替换 (Substitution)**：利用 re.sub() 将符合模式的旧文本替换为新内容，这是数据清洗的核心。
- **分割 (Split)**：根据复杂的正则模式将字符串拆分为列表。

- **异常处理**：在进行网络资源定位（URL）处理时，urllib.parse 会处理特殊的字符编码（如将中文转为 %xx 格式），这与正则清洗逻辑互补。

### 7.2 正则数据清洗
在现代工程（如 FastAPI 或大模型应用）中，数据清洗通常遵循 “提取 → 清洗 → 校验” 的自动化流水线：

#### 第一阶段：字符串预处理
- **格式清理**：利用字符串自带的方法去除首尾空格或统一大小写。
- **二进制转换**：处理图像或音频等二进制数据时，需先将 bytes 类型转换为 Unicode 字符串，或直接在字节流上应用正则。

#### 第二阶段：复杂正则匹配与替换
利用 re 模块处理非结构化文本中的噪声：
- **去除冗余**：使用正则匹配 HTML 标签、特殊符号或重复的空白字符并将其剔除。
- **模式提取**：从杂乱的日志或文档中精准提取日期、邮箱或自定义的业务 ID。

#### 第三阶段：工程化校验 (Pydantic 驱动)
正则清洗后的结果通常需要通过 Pydantic 进行最后的“门卫”检查：
- **Field 约束**：在 Pydantic 模型中，可以使用 regex（或新版本中的 pattern）参数直接为字段定义正则匹配规则。
- **自定义验证器 (Validator)**：如果正则逻辑过于复杂（涉及多字段依赖），可以编写自定义验证器函数。如果校验失败，Pydantic 会自动抛出清晰的 ValidationError 供接口返回。
- **数据类型强制转换**：Pydantic 会尝试将清洗后的字符串转换为定义的类型（如将 "99.5" 转换为 float），确保进入业务逻辑的数据是“干净且准确”的。
